# ⚡ Atomic Habits — 2026 Daily System

A beautiful, interactive daily habit tracker inspired by **Atomic Habits** by James Clear. Built for the full year 2026 with persistent data storage.

## ✨ Features

- **📅 Full Year 2026 Tracker** — Navigate any day from Jan 1 to Dec 31
- **💾 Persistent Storage** — All data saved in localStorage, survives browser restarts
- **✅ 11 Daily Time Blocks** — From 8 AM wake-up to 11 PM wind-down
- **🍽️ 7-Meal Calorie Tracker** — Targeting 3,200 cal/day for muscle gain
- **📊 Weekly Habit Grid** — Track 5 core habits across the week
- **🗓️ Year Heatmap** — GitHub-style contribution grid for the entire year
- **📝 Custom To-Do List** — Add/remove your own daily tasks
- **🧠 Atomic Habits System** — 4 Laws, habit stacking, and identity statements

## 🚀 How to Use

1. Open `atomic_habits_daily_system.html` in your browser
2. Tick off habits as you complete them throughout the day
3. Navigate between days using the arrow buttons
4. Add custom tasks in the "My Tasks" section
5. Check the Year tab to see your progress over time

## 🛠️ Tech Stack

- **HTML5** — Structure
- **CSS3** — Dark theme with gradient accents and micro-animations
- **Vanilla JavaScript** — Logic and localStorage persistence
- **Tabler Icons** — Beautiful icon set

## 🎯 The 4 Laws Applied

| Law | Application |
|-----|------------|
| **Make it Obvious** | Book on desk, gym clothes laid out, phone across room |
| **Make it Attractive** | Bundle coding with music, connect habits to identity |
| **Make it Easy** | Every habit has a 2-minute version |
| **Make it Satisfying** | Visual checkmarks and progress tracking |
